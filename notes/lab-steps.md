# Cloud Infrastructure Monitoring System

## Objective
Deploy and monitor a Linux server hosted in AWS.

## Environment
- AWS EC2
- Ubuntu Linux
- SSH

## Monitoring Tasks
- CPU usage
- Memory usage
- Disk utilization
- Network interfaces
- Open ports
- Connectivity testing

## Commands Used

```bash
htop
free -h
df -h
ip addr
ss -tulnp
```

## Outcome
Created a monitoring workflow to observe Linux server performance and troubleshoot network and resource issues.
